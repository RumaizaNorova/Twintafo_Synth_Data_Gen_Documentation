# Use cases

## Product development & testing
- develop analytics pipelines without waiting on real data access
- test ETL, schema migrations, and dashboards

## Benchmarking & method development
- prototype feature engineering and modeling approaches
- validate performance expectations on realistic distributions

## Education & demonstrations (public-safe)
- demonstrate workflows without exposing real patient data

!!! warning
    Synthetic data should not be used for direct clinical decision-making unless explicitly validated and authorized for that use.
